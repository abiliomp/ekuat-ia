# Guías — Índice

Esta sección contiene guías prácticas para la integración y uso del Sistema Integrado de Facturación Electrónica Nacional (SIFEN / e-kuatia).

## Guías disponibles

| Guía | Descripción | Audiencia |
|---|---|---|
| [Guía de Pruebas](./guia-de-pruebas.md) | Condiciones, set de datos y secuencia de pruebas mínimas en el ambiente de test del SIFEN | Equipos de integración, QA, facturadores electrónicos |
| [Mejores Prácticas — Envío de DE](./mejores-practicas-envio-de.md) | Recomendaciones para generación de XML, gestión de lotes, uso de los WS asíncronos (recibe-lote, consulta-lote, consulta), motivos de rechazo y bloqueo | Desarrolladores de software de facturación |
