# Liquidación de Crédito Electrónica (LCE)

> **Fuente:** Manual Técnico SIFEN v150, campo C002

## Identificación

| Atributo | Valor |
|----------|-------|
| **Código C002** | 3 |
| **Descripción C003** | "Factura electrónica de importación" (según MT v150) |
| **Estado** | Futuro — No implementado en versión actual |

## Descripción

La Liquidación de Crédito Electrónica está contemplada en el Manual Técnico v150 como el tipo de documento 3. Su implementación es futura y aún no se encuentran definidos todos los campos específicos en el Schema XML actual.

## Estado Actual

Este tipo de documento electrónico no está operativo en el SIFEN en la versión actual del Manual Técnico v150. Los campos específicos del grupo XML correspondiente (E200-E299 para Factura de Importación, según la tabla de grupos del MT) serán definidos en versiones futuras.

## Nota sobre Nomenclatura

El Manual Técnico v150 en la tabla del campo C002 lista:
- Código 2: Factura electrónica de exportación (Futuro)
- Código 3: Factura electrónica de importación (Futuro)

La denominación "Liquidación de Crédito Electrónica (LCE)" con código 3 puede corresponder a una denominación alternativa. Se recomienda verificar las Notas Técnicas más recientes y la documentación actualizada de la DNIT para confirmar el estado de implementación.
